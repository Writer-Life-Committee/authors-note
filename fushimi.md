# 伏見瞬

# 担当する目次
7.選考：基準／システム／書評／選評／批評／受賞

# 主題
「ズレ」が「ズレ」でなくなる時

# プロフィール
文筆の悪魔

# 草案
日常と書いたもの（自分の場合は主に批評）のズレ、原稿料の非均等性（努力と対価がほとんど一致しないこと）、書物から得る情報に対する（謎の）信頼感、体験談など

# 依頼概要
膨大な先行投資を要する装置産業とはちがい、文筆業・編集者業は、知恵と技術と人脈さえあれば、独力でも仕事をつくり、収益を生み出せる意味で、「稼ぎやすい仕事」です。

しかしその分、参入障壁は低く、読者は移り気で、安定収益を見込みづらく、ともすれば一極集中の人気商売になりがちな、「稼ぎづらい仕事」でもあります。

求められる職能も多岐にわたり、隣接業種との――たとえば企画、編集、制作、デザイン――棲み分けもゆるやかで、このことは歴史上に多くの功罪を生み出してきました。

たとえば、日本語による文章表現の歴史には、性風俗の案内・評判記から出発し、大人向けコミックや長編フィクション、イラスト集の企画出版で頭角を現して、多くの才能を芸術界に輩出した男がいます。文学青年から総合誌の編集長、出版事業の経営者を経てメディア産業の重鎮となり、最後は戦争協力に加担した男もいます。

彼らの生涯を費やした栄枯盛衰が私たちに伝えるのは、「読み、書くこと」とは、たくさんの声、視線、立場がややこしく絡み合うこの世界で、あたかも「そこに言葉だけがある」かのようにふるまう、本来的に無数の「ずれ」を孕んだ営みなのだろうということです。

不思議、不似合い、不格好。不確か、不手際、不釣り合い。打ち消しの接頭辞と常に親しむこの職種で、時には執筆し、時には編集し、時には経営と呼ぶべき立場を行き来する日々には、どのような悩ましさがあり、どういった楽しみがあるのか。そのいくつかをふり返り、ひも解くことで、ひとりの若い出版者が実践する選択と考案のポリシー――言い換えれば、批評家としての本懐――を、（ほんの少しだけ）垣間見せてほしいです。

# 草稿
## 1.

　おそらく、この原稿で私に求められているものは、雑誌の編集長の立場から文筆の営みの複雑さを描くことだろう。編集、執筆、経営を兼ねる、つまり雑誌全体の流れを調整する（編集）、手と頭を労使して文章を生成する（執筆）、金銭の動きをコントロールする（経営）、という異なる目的地を持った作業が同時進行するのが雑誌作り。それぞれの作業で関わる人が違うわけだから、それぞれに気を配らなくてはいけない。幾分かはストレッシブな、幾分かはエキサイティングな複雑な営みをどのように行って、どのような苦労や歓びを得ているのか。

　正直言うと、「なんとなくやっている」以上の答えが出てこない。私自身の生き方がそんな感じだから。自分の楽しみと周囲の状況を上手く擦り合わせたいし（編集？）、なにかしらの作品作りに精魂を注ぎたいし（執筆？）、お金を稼いで買うべきものを買いまくりつつ生活を安定させたい（経営？）。それぞれの欲望は多くの人に当てはまるかもしれないが、私は３つをどれも手放さずに全部叶えたいわがままな性格を今日に至るまで持ち続けた。異なるゴールをいくつも抱えたまま生活するのが当たり前になっていて、どうやって実現しているかなんてわからない。人生の通常運転の一部なので特筆すべき苦労も歓びもなく、自分自身の普通に忠実でいたら今みたいになっただけ。そう言う意味では、複数の作業を同時並行する今の立場はめちゃくちゃ向いているのかもしれない。

　上記の理由で、暗に要請されているであろうお題に応えることができない。代わりになるか心許ないけれど、私の現時点での悩みについて書いてみる。端的に、やる気が出ないのだ。

## 2.

　『LOCUST』という雑誌を十数名の友人とともに出版している。完全な自費出版だ。

　『LOCUST』は旅行と批評をかけあわせた本である。毎回、特定の場所に集団で旅行に行き、互いの考えや感じ方を交換した上で、批評文やエッセイを執筆し、互いに編集する。2018年からこれまでに4冊を刊行し、この秋には新刊を出す予定だ。自分たちで誇れる本が作れてきたと感じているし、売れ行きもリアクションも予想外に大きかった。手応えのある活動ができていることは単純に嬉しいし、その手応えが次のモチベーションにつながるという良いサイクルもできている。そう感じていた。　

　しかし、ここ数ヶ月、ずっとモヤモヤしていた。LOCUSTが上手くいっている実感がどうにも持てないのだ。最初は単純に作り続けることのマンネリ感や、他の原稿を抱えているせいだと思っていた。それもあるが、最近原因が別のところにあることに気づいた。


　私は「ズレ」に惹かれ続けた人間だ。自分が編集長を務めている雑誌でも、「ズレ」を意識して作ってきた。今、その「ズレ」が機能しなくなってきている。

　私も以前に出演したSpotifyのpodcast番組「pop life the podcast」のある回で、「雑誌が軒並みに部数を落としている中、文芸誌だけがここ数年部数を上げている」という話がでていた。実際、『文藝』『群像』といった老舗の文芸誌がリニューアルを行い、特集の組み方や論点の提示を工夫し、増刷やネット書店での売り切れが発生するなど、盛況を見せている。podcastの放送の中では、ホストの田中宗一郎氏がその状況を「イデオロギーが強く、ライフスタイルが弱い時代」と分析していた。ある種の思想・イデオロギーを伝える文芸誌が活躍し、生活や流行に根ざしたライフスタイルを提示する雑誌が衰退している。この分析の是非は置いといて（私は正しいと思う）、個人的に重要なのは、ここで言う「イデオロギー」と「ライフスタイル」の対立項が、私にとってどちらも愛着の対象で、それが『LOCUST』という雑誌のコンセプトに大きく関わっている点だ。

## 3.

　『LOCUST』が旅行をテーマにするきっかけは半ば偶然だ。もともとこの雑誌の制作は、「ゲンロン批評再生塾」という批評を学ぶ集まりの、第3期の同期生によってはじまった。先達にあたる人々の多くが同人誌を作っており、「うちらもやるっしょ」という空気が自然とできていた。ただし、批評に興味があるという点を除いては、参加者はバックグラウンドをほとんど共有していない。得意とする分野は音楽、映画、演劇、美術、文学、都市論などバラバラ。職業もバラバラで、学生もいれば、医者や学習塾経営者もいる。そこで、浮かんできたアイディアが「旅行」だった。旅行に行って、その土地に関わる本や映画などを取り上げれば、ジャンルが異なっていても一本の筋は通せる。そう思って始めた企画だった。もともと旅行自体に興味があったわけではない。

　ただ、私が旅行というアイディアに興奮していたのは、「ズレ」を生む力がそこにあったから。一般的に旅行は「ライフスタイル」に属する行いだろう。旅行は一つのレジャー、多くの現代人が趣味の一つとして認識しているものだし、旅行や旅行誌に「イデオロギー」を求める人はいないだろう。言い方を変えれば、旅行の本にシリアスな社会批評や哲学論文は求められていないだろう。批評再生塾に集まってLOCUSTを作り始めたメンバーは明らかに批評や哲学や芸術に惹かれていた。書く文章だって、文芸誌に乗るタイプのスタイルを有している。だから「ライフスタイル」側の旅行誌のふりをして、「イデオロギー」側の批評的な文章を読ませると、面白い「ズレ」が生まれると思った。

　一番こだわりたかったのはデザインだった。一般に流通している文芸誌や批評誌のスタイル、白黒で縦書きの文字が並んでいる形式だとつまらない。むしろその逆、カラーで横書きに刷るべきだと感じていた。多くの旅行誌、カルチャー誌、ファッション誌のスタイル。「ライフスタイル」型の形式を採用するのは絶対に必要だと思って、反対意見を説得して押し通した。「ライフスタイル」を求めて買った人が「イデオロギー」を発見する、あるいは「イデオロギー」を求めてた人が「ライフスタイル」を発見する。期待していたものからの「ズレ」によって、読者に豊かな体験がもたらされる。それぞれの立場にとっての新鮮さが生まれると思ったし、個人的には学生の時からその両方が色濃く含まれている雑誌があればいいのにとなんとなく考えていた。以上のような理屈も当時は意識していなくて、とにかく横書きカラーの方がいい気がするからそっちでいかせてくれとお願いした記憶がある。当時は普通と違うことがしたいくらいの考えだった。

　とはいえ、アイディアだけあっても実現はできない。『LOCUST』は好運なことに、優れたデザイナーと出会えた。副編集長の古くからの友人がデザインをやっていて、彼のチームに依頼をしている。結果できあがったものは、自分が想定していたより力のある、他に類をみないデザイン。vol.3以降は、各原稿それぞれに合わせたデザインを形にしてくれて、より強力な本になった。デザインに関してはもっと四苦八苦するだろうと予期していたから、早い段階で高い水準を達成できたのはとてつもなくラッキーだった。

## 4.

　実際、LOCUSTはさまざまなタイプの人が興味を持ってくれる雑誌になったと思う。発行部数以上に、特定のグループ・クラスタを越えて読まれることが誇りだった。「イデオロギー」と「ライフスタイル」の間の「ズレ」を活かす戦略は、上手く機能していた。　　しかし、コロナ禍以降、その掛け合わせが難しくなってしまった。「旅行」が気軽にできるライフスタイルの一部ではなくなってしまったからだ。むしろ、「旅行」はイデオロギー的な行為になっている。旅行することはリスクを伴う営みであり、その実践は思想的なメッセージの表明を内包している。人が集まり、移動していくこと（集会の自由、移動の自由）には大きな意味があり、その実践を止めてはいけない。私も実際そのように考えているし、LOCUSTの活動も、人が集まり移動することの意味を雑誌として物質化するからこそ価値がある。しかし、そこに「ライフスタイル」的な、生活の一部として気軽に楽しめる旅行はない。かつて「旅行誌」にあった、軽薄さや気安さが失われてしまったのだ。 　 　

　元のコンセプトに忠実であるなら、「旅行」に変わるライフスタイルの提示の要素を、LOCUSTに加えなくてはいけないだろう。今のところ具体的なことは思いついていない。極端な話、「旅行」をやめて別のなにかをはじめるかもしれない。 11月には新刊が出るが、今回はその途中経過のような文章になった。ただ、「ズレ」が生じていない状況に気づいて、私のやる気はかなり復活した。状況の認識はいつでも良い治療法だ。 

 　私は「ズレを作る立場からズレてはいけない」と直感している。個人的には、ズレを意識していたほうが単に生きてくのが楽しいってだけなのだけど、文筆という営みそのものもズレたままでいることを肝要としているだろう。それはあえて捻くれた立場を取ることとは違う。そもそも私たちは誰もがズレているのであって、社会にはズレを消しさる作用が強烈にある。その作用とは別の力を生み出すこと。それは、「素直」であることに近い。ズレを生み続けるということは、言い換えれば素直であるということだ。その人そのものであるということだ。 　

　保坂和志が『小説の自由』と言う本の中で、カポーティ『冷血』とドストエフスキー『罪と罰』について語りながら、「生きる目的は彼が誰でもなく彼自身になること」というテーゼを描き出していた。正しい発想だ。私たちにはズレながら生きる権利がある。そして文筆は、創作にせよ批評にせよエッセイにせよ、「その人がその人自身になる」という目的を実現するための手段だ。文筆に関わる編集や営業は、「その人自身」を世の流れに流通するために存在している。個人と社会が正しくズレ続けるために、私たちは文筆を要望する。
