Where to get help

If you think you’ve found a bug in the Merlot theme, please check the existing issues, and if no one has reported the problem, open a new issue.

If you have a general question about the theme, how to implement it, or how to customize it for your site you have two options:

  1. Search for your query on support.github.com, which will also look for similar topics on github.community
  2. Ask your question of the Jekyll community on talk.jekyllrb.com
  3. Contact GitHub Support
